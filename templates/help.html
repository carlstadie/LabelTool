Overview

{{ cfg.app_name }} is a local-network remote sensing image labelling platform developed for the Permafrost Remote Sensing section of the Alfred Wegener Institute. It supports GeoTIFF image uploads, multi-class polygon/rectangle annotation, and export to multiple geospatial formats.

The platform runs on a local server and provides two separate access points:

  • Admin Portal — http://<server>:5000/admin
  • Worker Portal — http://<server>:5000/worker

Admin Portal

Log in with the Admin Token (default: awi-admin-2024, can be changed in Admin Config).

Groups

Organise images into groups by project, region, or sensor. Create groups before uploading images.

Upload Images

Drag & drop or browse for GeoTIFF files (.tif / .tiff). Images must be georeferenced. Upon upload, metadata (CRS, resolution, band count, bounding box) is extracted and a preview PNG is generated. Large images are resampled for display.

Label Classes

Define classes with a key number (1–9), name, colour, and description. Workers press the corresponding number key to assign the class to a drawn shape. Default classes are provided for common permafrost features.

Configuration

Set the application name, subtitle, organisation line, logo URL, admin token, and default tile size.

Worker Portal

Workers enter their name (no password required) at login. The name is recorded with every label created. The image queue shows all available images. Clicking Label → opens the labelling tool and marks the image as in progress.

Labelling Tool

Ribbon Tools

  • Pan — Navigate the image (drag to pan, scroll to zoom)
  • Rectangle — Click and drag to draw a bounding box (like QGIS)
  • Polygon — Click vertices; double-click to close
  • Delete — Click any label to remove it
  • Grid — Toggle tile grid overlay

Tile Grid

The grid divides the image into tiles of the specified size in metres. Adjust the slider in the ribbon (10 m – 2000 m). The grid is redrawn instantly and is stored per-label for provenance.

Band Order

Use the R / G / B dropdowns in the ribbon to change which bands are mapped to red, green, and blue. The image preview re-renders after a short delay. Useful for false-colour composites on multi-band imagery.

Assigning Classes

After you finish drawing a shape, a class picker automatically appears. Select a class by clicking it or pressing its corresponding number key (0–9). Key 0 opens the Other flow: enter a new class name or type an existing class name/number.

Saving

Labels are saved to the server immediately after creation. You can still click Save or press S to retry any pending unsaved labels; an auto-save retry also runs every 2 minutes. Dashed borders indicate unsaved labels; solid borders indicate saved ones.

Finishing

Use Close (or C) to save your current labels and end your session while returning the image to available. Use Complete (or Enter) to save all labels and mark the image as done.

Export Formats

Both the Admin and Worker portals offer export of labels and image footprints with optional filtering by group, class, and (for workers) by own labels.

FormatExtensionUse Case
GeoJSON.geojsonWeb mapping, QGIS, universal
Shapefile.zipArcGIS, QGIS, legacy workflows
GeoPackage.gpkgQGIS, single-file, open standard
GeoParquet.parquetBig data / ML pipelines, pandas/geopandas
CSV (WKT).csvTabular analysis, spreadsheets

Image Footprints export the bounding boxes of images with a summary of which classes appear in each image — useful for scene-level filtering and dataset inventories.

Setup & Installation

# 1. Create a virtual environment
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run the server
python app.py

# Admin:  http://localhost:5000/admin
# Worker: http://localhost:5000/worker
# Default admin token: awi-admin-2024

For network access, replace localhost with the server's LAN IP address. Workers connect from any browser on the same network.
